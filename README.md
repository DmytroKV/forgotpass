# Forgot Password Website

A simple webpage with an animated "Forgot Password?" title in the center, and scrollable content.

## Usage

Open `index.html` in a web browser to view the page.

## Files

- `index.html`: The main HTML file with the animated title and content.
- `style.css`: CSS file for styling and animation.
- `script.js`: JavaScript file for interactivity (showing the message box).

## Features

- Centered animated title that fades and scales.
- Scrollable page with placeholder content.
- "Reset Password" button that reveals a message box.
- Form to send a message via email (opens default email client).

## Troubleshooting

## Troubleshooting

- Ensure both `index.html` and `style.css` are in the same directory.
- If animations don't work, check browser compatibility.
